Introdução À tag <img>

A tag img em HTML é usada para incorporar páginas web. ela é um elementpo vazio, ou seja não possui uma tag de fechamento o contem apenas atributos que defines o conteudo visual

Atributos principais

  • src (source): define o caminho da imagem que será exibida pode ser uma URL externa ou um caminho relativo dentro do servidor.
