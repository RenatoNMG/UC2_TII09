Introdução À tag <img>

A tag img <img>em HTML é usada para incorporar páginas web. ela é um elementpo vazio, ou seja não possui uma tag de fechamento o contem apenas atributos que defines o conteudo visual

Atributos principais

  • src (source): define o caminho da imagem que será exibigda. Pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1:

    <img src="image1.jpg">

    Resultado:

  • Alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. Ele também melhora a acessibilidade para leitores de tela.

    Exemplo 2

    <img src="imagegif.gif" alt="Gato gif" style="width: 100%">

    Resultado

  • src Tambem: pode ser de um link externo

    Exemplo 3:

    <img src="https://http.cat/images/425.jpg" alt="" style="width: 100%>

    Resultado

Tipos comuns de imagens

Os formatos de imagens mais utiliazasos na web são

  • (.jpg ou .jpeg) Formayo utilizado amplamente para fotografia e imagens com muitos detalhes e cores. possui compressão com perda de qualidad (Lossy).
  • (.png) Usdo par imagens com fundo trasparente ou que necessita de alta qualidade,sem perda de detalhes. utiliza compressão sem perda (lossless)
  • GIF (.gif) Formato ideal para imagens animadas e pequenos ícones. Suporta transparência, mas tem um número limitdo de cores.
