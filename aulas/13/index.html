Estrutura basica do Grid

para criar um layout Grid, primeiro evemos definir um containder que ira funcionar como uma grade. dentro desse conteiner, os itens se alinha conforme as colunas e linhas definidas. 

            
    div class="grid-container">
    div class="item item1">item 1/div
    div class="item item2">item 2/div
    div class="item item3 bordaredonda">item 3/div
    div class="item item4">item 4/div
    /div

    .grid-container {
    display: grid; /* Define o container como grid */
    grid-template-columns: 200px 200px; /* Define duas colunas de 200px cada */
    grid-template-rows: 150px 150px;
    gap: 10px;
    background-color: lightgray;
    }

Ajustando Colunas e linhas com o fr

unidde fr (fraction unit) é muito util se deseja criar colunas ou linhas proporcionais, distribuindo o espaço disponivel entre os itens.
