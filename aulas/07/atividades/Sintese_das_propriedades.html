Síntese das Propriedades CSS

  1. color

    2. Define a cor de um elemento.

    h1 {
    color: blue;
    }
  2. text-align

    3. alinha o texto dentro de um elemento.

    • left: alinha o texto á esquerda (padrão).
    • center: centraliza o texto
    • right: alinha o texto á direita
    h1 {
    text-alingn
    }
  3. font-size

    4. Define o tamanho da fonte de um elemento.

    h1 {
    font-size 18px;
    }
  4. background-color

    5. Define a cor do fundo de um elemento

    h1 {
    background-color: yellow;
    }
  5. color (para botões)

    6. define a cor do texto de um elemento (usando um conjunto com background-color)

    h1 {
    color: black;
    }
