Propriedade position no CSS e como Posicionar Elementos

introdução

no CSS propriedade position é usada para definir como os elementos serão posicionados ma página. ela oferece controle sobre o posicionamento permitindo criar leyouts ,ais dinamicos e precisos.

  • Static
    • Definição: é o valor pdrão. o elemento ´w posicionado de acordo com o fluxo da pagina
    • Exemplo 
                              
div{ 
psition: static
}
  • Relative
    • Definição: O elemento é posicionado em relação à sua posição original no fluxo da página. Utiliza as propriedades top, right, bottom e left para ajustar o deslocamento a partir da posição inicial.
    • Exemplo: 
                                      

.box {
    position: relative;
    top: 20px /* move 20 px para baixo da posição original */
    left: 30pc; /* move 30px  para direita */
}
  • Absolute
    • Definição: O elemento é fixado em relação ao seu containder pai mais proximo que tenha a propriedae positiondiferente de static.
    •                         
.parent {
    position: relative;

    .child {
        position: absolute;
        top:10px; /*move para baixo da posição original*/
        
    }
}
  • Fixed
    • Definição: O elemento fixado em uma posição especifica da janala (viewport) e permaneçe no mesmo lugar meso ao rolar a pagina
    • Exemplo: 
                              
     .box {
            position: fixed;
            top:0;
            right: 0;
     }
